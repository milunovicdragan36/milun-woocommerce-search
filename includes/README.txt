AJAX-powered WooCommerce product search with live results, category filtering, and flexible display positions.

== Description ==
Milun Search for WooCommerce adds advanced search features to your WooCommerce store, allowing searching by products, categories without refreshing the page.

== Installation ==
1. Upload the plugin folder to `/wp-content/plugins/`.
2. Activate the plugin via the 'Plugins' menu in WordPress.
3. Click on "Update" button and your searching form is ready to use.
